## Netguru recruitment task - react

**React app for movies management.**
Avaliable online **[here](https://getriax.herokuapp.com/)**
___
>  To run locally

		$ git clone https://github.com/Getriax/Netguru-task.git
		$ cd public/
		$ npm install
		$ npm start

___
> To build

		$ npm run build
The build will be in ../dist direcotry
___
> To test

		$ npm test
___
